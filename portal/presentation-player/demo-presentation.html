核心設計原則

  • 💪 獨立無依賴：採用原生 Vanilla JS，不需 React / Vue 或 Webpack。
  • 🎨 CSS 變數：顏色與字體全部由 CSS 變數控制，客製化非常簡單。
  • ⚡ 事件驅動：內建豐富的事件掛鉤 (slideChange, load, ended)。
  • 📦 獨立可複製：放在一個資料夾中，直接複製即可在其他專案運作。

如何複製本模組？

  1. 複製 portal/presentation-player/ 目錄到你的專案中。
  2. 在 HTML 引入 presentation-player.css 與 presentation-player.js。
  3. 在你的 HTML 建立一個播放器的掛載容器 <div id="player"></div>。
  4. 使用 JavaScript 初始化：const player = new PresentationPlayer({ container: document.getElementById('player') });
  5. 呼叫 player.loadFromUrl('your-slide.html')。
